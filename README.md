# 『ふたり反省会』
サイトへのリンク→

<em>就寝前の不毛な反省会にお供のひつじを</em><p>ネガティブに認知が歪みがちな就寝前の考え事をサポートする音声チャットwebアプリ。就寝前にPCはふさわしくないという考えから"in"flexibleデザインを採用。（スマホのみに対応したデザインにした。）PWA化しており、ホーム画面へのアイコン追加など、ネイティブアプリのように利用できる。</p>

## 技術仕様

### 1. 使用技術・言語

本プロジェクトは、主に以下の言語で構成されています。

| 言語 | 構成比 | 備考 |
| :--- | :--- | :--- |
| **HTML** | 73.3% | フロントエンドの構造定義 |
| **CSS** | 10.6% | フロントエンドのスタイリング |
| **JavaScript** | 8.5% | フロントエンドの動的処理、音声チャット機能の実現 |
| **Python** | 7.6% | バックエンドのサーバーロジック、外部ライブラリの利用 |

### 2. 外部ライブラリ・フレームワーク

| 種類 | 名称 | 備考 |
| :--- | :--- | :--- |
| **外部ライブラリ** | Flask | Pythonの軽量Webアプリケーションフレームワーク | |

### 3. 主要ファイルと構成要素

プロジェクトの構成要素と役割は以下の通りです。

| ファイル名 | 推測される役割 |
| :--- | :--- |
| **`app.py`** | アプリケーションのメイン実行ファイル（Python、Flaskサーバーのコアロジック） | |
| **`requirements.txt`** | 依存関係ライブラリ一覧（Pythonパッケージ管理） | |
| **`render.yaml`** | デプロイメント設定ファイル（Render(PaaS環境)向け設定） | |
| **`manifest.json`** | アプリケーションマニフェストファイル（PWA関連） | |
| **`service-worker.js`** | サービスワーカー（オフライン機能、プッシュ通知などのクライアントサイド機能） | |
| **`README.md`** | プロジェクトの説明書 | |
